# simple-mvvm-framework

This is a simple implemention of an MVVM framework which mainly mocks how Vue works.

一个MVVM 的简化版实现方案，主要参照Vue的实现原理

## install

```
npm install
```

## Usage

1. Open browser
2. visit index.html

## Features

1. Use {{ your_ varible }}, v-text, v-model in a template
2. Use data
3. Use methods
